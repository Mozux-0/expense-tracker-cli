# Expense Tracker CLI

A simple, command-line expense tracker built with Node.js that helps you manage your finances right from your terminal. Track expenses, view summaries, and manage your budget without leaving the command line.

![CLI Demo](https://img.shields.io/badge/CLI-Expense%20Tracker-blue)
![Node Version](https://img.shields.io/badge/node-%3E%3D16.0.0-brightgreen)
![License](https://img.shields.io/badge/license-MIT-green)

## Features

- **Add Expenses**: Quickly log expenses with description and amount
- **Delete Expenses**: Remove expenses by ID
- **View All Expenses**: List all expenses in a clean, readable format
- **Monthly Summary**: Get total expenses for any month
- **Persistent Storage**: All data saved to a local JSON file
- **Error Handling**: Robust validation and informative error messages
- **Colorful Output**: Easy-to-read colored terminal output
